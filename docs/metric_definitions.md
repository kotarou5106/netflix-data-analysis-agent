# Metric Definitions / 指标定义

These definitions describe the analysis scope used in this Netflix Data Analysis Agent project. They are project-specific assumptions and should not be presented as universal industry standards.

这些定义用于说明 Netflix Data Analysis Agent 项目中的分析口径。它们是本项目的分析假设，不应被表述为通用行业标准。

The metrics in this project support observational analysis only. They can help identify patterns, differences, and risk signals, but they cannot directly prove causality.

本项目中的指标只能支持观察性分析。它们可以帮助识别模式、差异和风险信号，但不能直接证明因果关系。

## Churn Rate / 流失率

English: Churn Rate measures the share of users marked as churned within a selected group or time scope in this project dataset.

中文：流失率表示在本项目数据集中，某个选定用户群体或分析范围内被标记为流失用户的比例。

## Completion Rate / 完播率

English: Completion Rate describes the percentage of watched content that users complete according to the dataset field used in this project.

中文：完播率表示用户观看内容的完成比例，具体含义以本项目数据集中的字段口径为准。

## Recommendation Click Rate / 推荐点击率

English: Recommendation Click Rate measures how often users click recommended content according to the recommendation interaction field in this project dataset.

中文：推荐点击率表示用户点击推荐内容的频率，基于本项目数据集中的推荐互动字段进行分析。

## Weekly Watch Hours / 周观看时长

English: Weekly Watch Hours estimates user viewing time per week. In this project, it can be derived from average watch time and weekly watch session fields when needed.

中文：周观看时长用于估算用户每周观看时间。在本项目中，可根据平均观看时长和每周观看次数等字段按需推导。

## Days Since Last Login / 距离上次登录天数

English: Days Since Last Login measures how many days have passed since a user's most recent login in the dataset.

中文：距离上次登录天数表示数据集中记录的用户最近一次登录距今的天数。

## Subscription Type / 订阅类型

English: Subscription Type represents the user's plan category in this project dataset, such as Basic, Standard, or Premium when those values are present.

中文：订阅类型表示本项目数据集中用户所属的套餐类别，例如数据中存在的 Basic、Standard 或 Premium。
